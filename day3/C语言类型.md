#### C语言类型分类

1. 整数
`char`,`short`,`int`,`long`,`long long`

char: 1个字节

short: 2个字节

int: 取决于编译器和CPU，通常的意义是'一个字'

long: 取决于编译器CPU，通常的意义是'一个字'

long long: 8个字节

*一个字长，指计算机寄存器的宽度，通常是32bit或者64bit*

整数的输入输出：
%d int
%u unsigned
%ld long long
%lu unsigned long long

2. 浮点数
`float`,`double`,`long double`
3. 逻辑
`bool`
4. 指针
5. 自定义
